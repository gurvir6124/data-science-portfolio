# data-science-portfolio

This repository contains 3 projects completed during my Physics degree at Imperial College London, with a focus on computational modelling, statistics, and machine learning.

## Projects

### 1. Particle Collision Simulation
A Python-based simulation modelling particle motion in a circular container using an event-driven approach. The project simulates elastic collisions and links microscopic particle behaviour to macroscopic thermodynamic properties such as pressure and kinetic energy.

### 2. Chi-Squared Analysis
Application of chi-squared goodness-of-fit testing to binned particle energy data, demonstrating statistical analysis and hypothesis testing.

### 3. Decision Tree Classification
Implementation of a classification model using decision trees to identify patterns and make predictions from data.

## Skills Demonstrated
- Python programming
- Object-oriented programming
- Statistical analysis and hypothesis testing
- Computational modelling
- Machine learning fundamentals
